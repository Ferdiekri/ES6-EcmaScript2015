Variables: let & var

En este apartado vamos a ver la diferencia entre let y var.

La diferencia entre las sos es el alcance de las variables.

  • var: define una variable global o local en una función sin importar el ámbito del bloque.
  • let: permite declarar variables limitando su alcance al bloque, declaración, o expresión donde se está usando

        var a = 5;
        var b = 10;
         
        if (a === 5) {
          let a = 4; // El alcance es dentro del bloque if
          var b = 1; // El alcance es dentro de la función
         
          console.log(a);  // 4
          console.log(b);  // 1
        }
         
        console.log(a); // 5
        console.log(b); // 1
