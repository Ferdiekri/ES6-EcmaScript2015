Funciones Arrow

La expresión de función arrow tiene una sintaxis más corta que una expresión de función convencional y no vincula sus propios this, arguments, super o new.target.

Las funciones arrow siempre son anónimas. Estas funciones son funciones no relacionadas con métodos y no pueden ser usadas como constructores.


        var materials = [
            'Hydrogen',
            'Helium',
            'Lithium',
            'Beryllium'
            ];
          
        console.log(materials.map(material => material.length));
        // expected output: Array [8, 6, 7, 9]
